# 🚀 Workforce Analytics Dashboard

> A modern, interactive dashboard that transforms raw job market data into meaningful insights 📊✨

---

## 📌 About the Project

This project is a **Data Visualization Dashboard** designed to analyze the **Indian fresher job market** using real-world data.

It provides insights into:
- 💼 Job roles & demand
- 💰 Salary trends
- 🧠 Skills required in the market
- 📍 Location-based opportunities
- 📈 Hiring patterns & offer rates

Built using pure frontend technologies with powerful visualization libraries.

---

## 🎯 Objective

To simplify complex job market data into **interactive and easy-to-understand visual insights**, helping students and professionals understand industry trends.

---

## ✨ Key Features

- 📊 8+ Interactive Charts (Google Charts, CanvasJS, D3.js)
- ⚡ Smooth animations & responsive design
- 🎨 Modern Neon/Synthwave UI
- 📈 Real-time data visualization from CSV
- 📱 Fully responsive (Mobile + Desktop)
- 🔄 Auto-resizing charts

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling (Neon UI)  
- **JavaScript (ES6)** – Logic & Data Handling  

### 📊 Visualization Libraries:
- **D3.js** – Custom charts & CSV parsing  
- **Google Charts** – Pie & Column charts  
- **CanvasJS** – Interactive animated charts
